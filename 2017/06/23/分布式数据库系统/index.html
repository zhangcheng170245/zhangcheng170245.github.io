一.环境介绍

1.1. mysql节点1环境

  • 操作系统版本 : centos6.5 x64

  • mycat版本 ：1.3 release
  • 数据库名 : db1
  • hostname:c1
  • ip:192.168.58.11

  • 数据库名 : db2
